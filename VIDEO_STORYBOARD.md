# Student Portal Explainer Video - Storyboard & Script

**Duration:** 30-40 seconds  
**Style:** Clean, modern, professional UI animation  
**Color Palette:** Soft gradients, blue (#1a3a5c, #1d4e89) and white  
**Format:** Suitable for website hero section (autoplay, muted, loopable)

---

## Scene 1: Introduction (0-5s)

**Visual:**
- Modern university dashboard loading on a laptop screen
- Smooth fade-in animation
- Dashboard elements appear with subtle slide-in effects
- Blue gradient background matching landing page (#1a3a5c)

**Text Overlay:**
- **Headline:** "One Platform. Every Student Service."

- Animation: Fade in from bottom, slight scale effect

**Timing:**
- 0-2s: Dashboard loading animation
- 2-5s: Text appears and holds

## Technical Specifications

### Colors
- **Primary Background:** #1a3a5c (Deep blue)
- **Secondary Blue:** #1d4e89
- **Accent Green:** #4caf50 (for CTA and success states)
- **White:** #ffffff (for text and cards)
- **Gradients:** Soft blue gradients matching landing page

### Typography
- **Headlines:** Bold, sans-serif (Arial/Helvetica)
- **Body Text:** Regular weight, readable size
- **CTA Text:** Bold, uppercase with letter-spacing

### Animation Principles
- **Easing:** Smooth cubic-bezier curves
- **Duration:** 0.3s - 0.8s for most transitions
- **Stagger:** 0.1s delay between sequential elements
- **Scale:** Subtle scale effects (1.0 to 1.05)
- **Opacity:** Fade transitions (0 to 1)

### Export Settings
- **Format:** MP4 (H.264) or WebM (VP9)
- **Resolution:** 1920x1080 (Full HD) or 1280x720 (HD)
- **Frame Rate:** 30fps or 60fps
- **Bitrate:** 5-10 Mbps for web
- **Audio:** None (muted)
- **Loop:** Seamless loop from end to beginning

## Tools & Software Recommendations

1. **After Effects** (Adobe) - Professional animation
2. **Figma** - Design and prototype
3. **Lottie** - Export JSON animations for web
4. **CSS/JavaScript** - Web-based animation (see `video-hero.html`)
5. **Blender** - 3D elements if needed
6. **Premiere Pro** - Final editing and export

---

## Implementation Notes

- All animations should be smooth and professional
- Text should be readable on all screen sizes
- Maintain consistent spacing and alignment
- Use the same color scheme as the landing page
- Ensure seamless looping (Scene 7 → Scene 1)
- Optimize for web performance (compressed, efficient)
